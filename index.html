Summarize what Nixpiry does
✦ Gemini
Nixpiry is a smart near-expiry stock return manager built for independent chemists in tier-2 Indian cities. It handles:

· Return tracking before drugs expire
· FEFO inventory (First Expiry, First Out)
· Khata / credit ledger management
· 7 modules: Core, Khata, Stock, Scheme, Vault, Flow, Score

Built under HaiCLOP Labs using Flutter + Supabase. 🧪
